## 项目背景：

### 比赛数据集基于Google Cloud Platform 上的 Big Query 中提供的2016 年纽约黄色出租车旅行记录数据。该数据最初由纽约市出租车和豪华轿车委员会 (TLC)发布。为了这个操场比赛的目的，对数据进行了采样和清理。根据个人旅行属性，预测测试集中每次旅行的持续时间。
